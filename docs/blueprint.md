# **App Name**: StudyGuru

## Core Features:

- Dashboard: Dashboard providing a quick overview of courses, tasks, and study progress.
- Course Management: Clean display of active, completed, and available courses with progress tracking.
- Pomodoro Timer: Timer for managing study intervals.
- Daily Study Plan: Personalized daily plan based on scheduled courses, deadlines, and study habits.
- Task Management: Management and tracking of homework and tests.
- AI-Powered Resource Finder: Smart tool that intelligently curates a personalized set of links to online learning resources and relevant documentation.

## Style Guidelines:

- Primary color: Purple (#8E44AD) to evoke feelings of intelligence.
- Background color: Light gray (#F0F4F8) to keep the focus on the content
- Accent color: Lavender (#E0BBE4) to subtly draw attention.
- Clear and modern sans-serif fonts optimized for readability.
- Consistent use of line icons throughout the interface for a clean and modern look.
- Clean and structured layout with a focus on content hierarchy.
- Subtle transitions and animations for feedback and enhanced user experience.
